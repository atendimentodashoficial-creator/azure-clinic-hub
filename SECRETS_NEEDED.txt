=== SECRETS NECESSÁRIOS PARA CONFIGURAÇÃO ===
Configure estes secrets no painel do Supabase (Settings > Edge Functions > Secrets)

OBRIGATÓRIOS (já providos automaticamente pelo Supabase):
- SUPABASE_URL
- SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY
- SUPABASE_DB_URL

OBRIGATÓRIOS (configurar manualmente):
- OPENAI_API_KEY          → Chave da API OpenAI (usado em agente IA, RAG, relatórios)
- CRON_SECRET             → Token para autenticar chamadas de cron jobs
- LOVABLE_API_KEY         → Chave de API do Lovable (proxy AI)

INTEGRAÇÕES (configurar se usar):
- UAZAPI_ADMIN_TOKEN      → Token admin do servidor UAZapi (WhatsApp)
- UAZAPI_SERVER_URL       → URL do servidor UAZapi
- N8N_API_KEY             → Chave da API n8n (automações)
- RESEND_API_KEY          → Chave da API Resend (emails)
- GOOGLE_CLIENT_ID        → OAuth Google Calendar
- GOOGLE_CLIENT_SECRET    → OAuth Google Calendar
- GOOGLE_REDIRECT_URI     → Callback URL do Google Calendar

PUBLICÁVEIS (podem ficar no .env do frontend):
- VITE_SUPABASE_URL
- VITE_SUPABASE_PUBLISHABLE_KEY
- VITE_SUPABASE_PROJECT_ID
